A text based version of blackjack

This project was done to get more used to breaking down something established into its most basic parts and transform it into code

This was also a test for classes when programming

A challenge was finding a way to create cards rather than just having a random value between 1-11 be the cards.
Using lists and 2 for loops, I was able to lock the values for 52 playing cards without having to assign them individually, each with their own suit, number, and values?
